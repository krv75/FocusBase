# FocusBase

Telegram-бот-площадка для подбора фотостудии: клиент ищет студии по фильтрам и добавляет в избранное, фотостудия заполняет профиль и загружает портфолио (фото/видео), администратор управляет студиями/клиентами и модерирует жалобы/отзывы.

## Возможности

### Клиент
- Меню клиента и навигация по студиям
- Поиск студий по типу съёмки (фильтр)
- Просмотр портфолио студии (фото/видео)
- Добавление студий в избранное
- Оставить отзыв, отправить обращение/жалобу

### Фотостудия
- Меню фотостудии
- Загрузка работ (фото/видео) с описанием
- Редактирование профиля (название/описание/контакты/типы съёмки)
- Редактирование портфолио (листать, менять описание, удалять)
- Просмотр рейтинга и отзывов

### Администратор
- Панель администратора
- Управление студиями и клиентами (листинг, удаление)
- Модерация жалоб (одобрить/отклонить/удалить студию)
- Модерация отзывов (листать/удалять)

## Технологии
- Python 3.10+ (рекомендуется)
- aiogram 3.x
- PostgreSQL
- asyncpg (пул соединений)
- python-dotenv
